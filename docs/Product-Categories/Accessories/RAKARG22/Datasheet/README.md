### Features

- Frequency range: 3550~3700&nbsp;MHz
- Gain: 10&nbsp;dBi
- VSWR: ≤ 1.31
- Efficiency: ≥ 75%
- Antenna type: Dipole
- Radiation pattern: Omnidirectional
- Polarization: Vertical
- Connector type: N-type female
- IP67 rated
